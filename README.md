## 概要

気象情報を一覧できる Web サイトです。現在位置における数日先までの気象予報をまとめて確認できます。

![hiyori](./.images/hiyori-device.png)

### 表示される情報

1. 天気
1. 温度
1. 体感温度
1. 風速
1. 湿度
1. 視界
1. 気圧

## 開発情報

本プログラムは以下を主な基盤として使用し、構築されています。

|                      | 使用技術      |
| -------------------- | ------------- |
| プログラミング言語   | TypeScript    |
| フレームワーク       | React         |
| デザインテンプレート | MUI           |
| HTTP クライアント    | axios         |
| 多言語対応           | React i18next |
